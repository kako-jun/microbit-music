# micro:bit Music Player 開発者向けドキュメント

micro:bit 単体で動作する音楽プレーヤー。MicroPython。

```
microbit-music/
├── main.py    # メインプログラム（約146行）
└── docs/      # 仕様書
```

## 技術メモ

### 音階データ形式
- `'音名+オクターブ:長さ'` — 例: `'C#5:4'`
- 音名は `note[:-1]`、オクターブは `note[-1]` で分離
- 12平均律テーブルで index ベースの移調計算

### LED表示
- `col = idx * 5 // 12` で12音を5列に均等分配（列0-4すべて使用）
- `level = (idx % 3) + 1` で高さ1-3
- 表示するのはキー変更後の音（shifted note）

### ボタン入力
- A+B同時 → A単独 → B単独 の優先順で elif チェーン
- 50ms×16回ポーリングで長押し/短押しを判定（非ブロッキング）
- shift の範囲: -12 〜 +12（1オクターブ）

### 曲追加
- `songs` 辞書にキー（曲名）と値（音符リスト）を追加するだけ
- `song_names` は `list(songs.keys())` で自動更新
