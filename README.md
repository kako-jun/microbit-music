# micro:bit Music Player

micro:bit単体で動作する多機能音楽プレーヤー

## 機能

### 🎵 複数曲対応
- ドラゴンクエスト序曲（ファンファーレ＋メインテーマ）
- ファイナルファンタジー序曲（アルペジオ風）

### 🎮 操作方法

#### 曲切り替え
- **A長押し** → 前の曲へ
- **B長押し** → 次の曲へ

#### キー変更（移調）
- **A短押し** → 半音下げ
- **B短押し** → 半音上げ
- **A+B同時押し** → キーリセット

※ 曲切り替え時、キーは自動的にリセットされます

### 💡 LED表示
現在の音の高さをバー表示で視覚的に表現
- 低音 → 1段
- 高音 → 複数段

## 使い方

1. micro:bitに`main.py`を転送
2. 電源を入れると "Ready" と表示
3. ボタンで操作開始

## 曲の追加方法

`songs`辞書に新しい曲を追加するだけ：

```python
songs = {
    "DQ序曲": [...],
    "FF序曲": [...],
    "新曲名": [
        'C4:4', 'D4:4', 'E4:4', ...  # 音階:長さ
    ]
}
```

### 音階フォーマット
- 音名：`C`, `C#`, `D`, `D#`, `E`, `F`, `F#`, `G`, `G#`, `A`, `A#`, `B`
- オクターブ：`3`～`8`
- 長さ：`1`（短）～`8`（長）

例：`'C5:4'` = C5の音を4単位の長さで
