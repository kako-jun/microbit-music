# micro:bit Music Player

micro:bit単体で動作する多機能音楽プレーヤー。複数楽曲の管理とリアルタイム移調（キー変更）が可能。

## 特徴

- 複数曲の管理と切り替え
- 半音単位のリアルタイム移調
- LED表示で音高を可視化
- ボタンの長押し/短押しによる直感的操作

## 操作方法

| 操作 | 機能 |
|------|------|
| A長押し | 前の曲 |
| B長押し | 次の曲 |
| A短押し | キー下げ（半音） |
| B短押し | キー上げ（半音） |
| A+B同時 | キーリセット |

※ 曲切り替え時、キーは自動的にリセットされます

## 収録曲

- ドラゴンクエスト序曲（ファンファーレ＋メインテーマ）
- ファイナルファンタジー序曲（アルペジオ風アレンジ）

## セットアップ

1. micro:bit Python Editorを開く
2. `main.py`のコードをコピー
3. micro:bitに転送
4. 電源を入れると "Ready" と表示

## 曲の追加

`songs`辞書に新しい曲を追加：

```python
songs = {
    "曲名": [
        'C4:4', 'D4:4', 'E4:4', ...  # 音階:長さ
    ]
}
```

音階フォーマット: `音名+オクターブ:長さ`（例: `C5:4`）

## 動作環境

- micro:bit v1.x / v2.x
- スピーカー（内蔵または外部）
