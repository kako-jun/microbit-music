# micro:bit Music Player

BBC micro:bit で動作する、ゲーム音楽プレーヤー。ドラゴンクエスト序曲やファイナルファンタジー序曲を、リアルタイムにキー（調）を変えながら再生できる。5×5 LED マトリクスで音の高さを可視化。

## 特徴

- ゲーム音楽の再生（DQ序曲 / FF序曲）
- 半音単位のリアルタイムキー変更（±12半音 = 1オクターブ）
- 5×5 LED で音高をバー表示（12音を5列に分配）
- ボタンの長押し/短押しで曲切替とキー変更を兼用
- 曲追加は辞書に音符を書くだけ

## 操作方法

| 操作 | 機能 |
|------|------|
| **A短押し** | キー半音下げ |
| **B短押し** | キー半音上げ |
| **A長押し**（800ms、非ブロッキング） | 前の曲へ |
| **B長押し**（800ms、非ブロッキング） | 次の曲へ |
| **A+B同時押し** | キーリセット（原調に戻る） |

- 曲切替時、キーは自動リセット
- 演奏中にボタンを押すと曲を中断して操作を受け付ける
- 起動時に最初の曲名をスクロール表示

## 収録曲

| 曲名 | 元ゲーム | ノート数 |
|---|---|---|
| DQ序曲 | ドラゴンクエスト（序曲マーチ） | 32音 |
| FF序曲 | ファイナルファンタジー（プレリュード） | 32音 |

## セットアップ

1. [micro:bit Python Editor](https://python.microbit.org/) を開く
2. `main.py` の内容をエディタにコピー
3. micro:bit に転送（USB接続 → ダウンロード）
4. 電源を入れると曲名がスクロール表示されて演奏開始

## 曲の追加方法

`songs` 辞書に追加するだけ:

```python
songs = {
    "曲名": [
        'C4:4', 'D4:4', 'E4:2', 'G4:8',
        # 音名+オクターブ:長さ（4=四分音符、2=八分音符、8=全音符）
    ]
}
```

### 音名フォーマット

- 音名: `C`, `C#`, `D`, `D#`, `E`, `F`, `F#`, `G`, `G#`, `A`, `A#`, `B`
- オクターブ: `1`〜`8`
- 長さ: `1`=最短 〜 `8`=最長（micro:bit の `music.play` 仕様に準拠）
