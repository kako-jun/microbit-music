# 使い方ガイド

## セットアップ

1. BBC micro:bit を USB ケーブルで PC に接続する
2. `main.py` を micro:bit に転送する（Mu Editor、micro:bit Python Editor、または `uflash` コマンド）
3. 転送が完了すると自動的にプログラムが起動し、LED に「Ready」とスクロール表示される

## ボタン操作

| 操作 | 動作 |
|---|---|
| A 短押し | キーを半音下げる |
| B 短押し | キーを半音上げる |
| A 長押し（800ms） | 前の曲へ |
| B 長押し（800ms） | 次の曲へ |
| A + B 同時押し | キーをリセット（元の調に戻す） |

短押しと長押しの境界は 800ms です。ボタンを押して 800ms 以内に離せば短押し、押し続ければ長押しとして認識されます。

## 画面表示

### 通常演奏中

5x5 LED に音の高さがバー表示されます。音が変わるたびにパターンが変化します。

### 操作時のフィードバック

- **キー変更時**: 「Key: +2」「Key: -1」のように現在のシフト量がスクロール表示される
- **曲切り替え時**: 新しい曲名がスクロール表示される
- **キーリセット時**: 「Key Reset」とスクロール表示される

## 曲の追加方法

`main.py` の `songs` 辞書に新しいエントリを追加するだけです。

```python
songs = {
    "DQ序曲": [
        'G4:2', 'C5:2', ...
    ],
    "新しい曲": [
        'C4:4', 'D4:4', 'E4:4', ...
    ]
}
```

音符の書式は MicroPython の music モジュール準拠です。

- `C4:4` = ド（第4オクターブ）を4拍
- `R:2` = 2拍の休符
- 音名: C, C#, D, D#, E, F, F#, G, G#, A, A#, B
- オクターブ: 1〜8

## 注意点

- micro:bit のメモリは限られているため、あまり長い曲データを入れるとメモリ不足になる場合があります
- 曲名に日本語を使う場合、LED スクロールで正しく表示されないことがあります（micro:bit のフォント制約）
