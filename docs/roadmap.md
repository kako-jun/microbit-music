# ロードマップ

## 近い将来に届けたいこと

### 収録曲の拡充

ドラゴンクエストとファイナルファンタジー以外にも、ゲーム音楽の名曲を追加していきたいと考えています。単音でも映える曲、短いフレーズでも「あの曲だ」とわかる曲を優先的に選びます。メモリの制約があるため、1曲あたりのデータサイズを意識しながら、なるべく多くの曲を楽しめるバランスを探ります。

### テンポコントロール

曲の再生速度をボタン操作で変えられるようにします。ゆっくり聴きたいとき、軽快に聴きたいとき、同じ曲でもテンポが変わると印象が大きく変わります。移調と組み合わせれば、自分だけのアレンジを楽しめます。

### LED パターンの強化

現在のバー表示に加えて、曲のリズムに合わせた点滅パターンや、音の変化を波のように表現するアニメーションなど、複数のビジュアルモードを用意したいと考えています。見た目の楽しさが増すことで、デスクの上に置いておきたくなるデバイスに近づきます。

## もう少し先の展開

### 曲データの外部読み込み

曲データをソースコードに埋め込むのではなく、外部ファイルから読み込む仕組みを検討しています。micro:bit v2 のファイルシステムを活用すれば、プログラムを書き換えずに曲を追加・入れ替えできるようになります。

### 録音・ループ機能

演奏した音をその場で録音し、ループ再生する機能です。ゲーム音楽をBGMとして流しながら作業するような使い方ができます。micro:bit のメモリ制約との兼ね合いが課題ですが、短いループなら実現可能と考えています。

### 2台連携

2台の micro:bit を無線通信で接続し、片方がメロディ、もう片方が伴奏を担当する構成です。単音の制約を超えて、より豊かな演奏体験を届けます。micro:bit の radio モジュールを使えば、同期再生の仕組み自体は実装可能です。
