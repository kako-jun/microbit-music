# microbit-music — ゲーム音楽プレイヤー for BBC micro:bit

## アプリの概要

**microbit-music** は、BBC micro:bit でゲーム音楽を演奏する MicroPython プログラムです。ドラゴンクエスト序曲やファイナルファンタジー序曲など、誰もが知っている名曲を micro:bit のスピーカーから鳴らします。

## 解決したい課題

micro:bit は教育用マイコンとして広く使われているけれど、「ブザーで音を鳴らす」止まりの作例が多い。せっかく音楽ライブラリがあるのに、実際に「聴きたくなる曲」を演奏している例は少ない。

ゲーム音楽は限られた音源でも魅力的に聞こえるように作られている。micro:bit の単音スピーカーとの相性は抜群のはず。「この小さな基板からあの曲が流れる」という驚きと楽しさを届けたい。それが microbit-music を作った理由です。

## ビジョン

- **聴いて楽しい**。教材ではなく、音楽プレイヤーとして成立する体験を目指す
- **触って楽しい**。ボタン2つだけで選曲・キー変更・リセットがすべてできる
- **見て楽しい**。演奏中の音の高さが LED マトリクスにリアルタイムで光る
- **拡張しやすい**。曲データを辞書に追加するだけで新しい曲が増える

## 対象ユーザー

- micro:bit を持っていて、何か面白いことをしたい人
- ゲーム音楽が好きで、小さなデバイスから流れる音に心が躍る人
- MicroPython の実践的なサンプルを探している学習者
