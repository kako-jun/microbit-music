# 主要機能

## マルチソング対応

複数の曲を内蔵しており、ボタン操作で切り替えられます。現在はドラゴンクエスト序曲とファイナルファンタジー序曲を収録しています。曲データは Python の辞書として管理されており、新しい曲の追加は配列を書き足すだけで完了します。

曲を切り替えると、曲名が LED にスクロール表示されます。

## リアルタイム移調（キートランスポーズ）

演奏中のキー（調）をボタンひとつで半音単位で上げ下げできます。同じ曲でも明るく聞こえたり、落ち着いて聞こえたりと、印象が変わる体験を楽しめます。

移調はオクターブの境界を自動的に処理するため、極端にキーを上げ下げしても音が破綻しません。キーの変更量は LED にスクロール表示されます。

## LED ビジュアライゼーション

演奏中の音の高さに連動して、5x5 LED マトリクスがリアルタイムに光ります。12音階を5列に割り当て、下から上へバーが伸びるように表示します。音楽に合わせて LED が動く様子は、見ているだけでも楽しいものです。

## 2ボタンによるフルコントロール

micro:bit の A ボタンと B ボタンだけで、すべての操作を実現しています。短押しと長押し（800ms）の区別、同時押しの3パターンを組み合わせることで、5つの操作を2つのボタンに収めています。

## 曲間の途中操作

曲の再生中でもボタン入力を受け付けます。長い曲の途中で次の曲に切り替えたり、キーを変えたりできます。演奏が終わるまで待つ必要はありません。
