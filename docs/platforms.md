# 対応環境

## 対応ハードウェア

| ハードウェア | サポート状況 |
|---|---|
| BBC micro:bit v1 | 対応（外部スピーカー接続が必要） |
| BBC micro:bit v2 | 対応（内蔵スピーカーで再生可能） |

micro:bit v1 にはスピーカーが内蔵されていないため、P0 ピンとGND に外部スピーカーまたは圧電ブザーを接続する必要があります。v2 は内蔵スピーカーがあるため、基板単体で音が鳴ります。

## 開発環境

| 環境 | 対応 |
|---|---|
| MicroPython | 対応（標準の music モジュールを使用） |
| MakeCode（ブロック） | 非対応（Python 専用） |

### 転送方法

micro:bit への転送は以下のいずれかで行えます。

- **Mu Editor**: micro:bit モードで「転送」ボタンを押す
- **micro:bit Python Editor**（Web）: ブラウザから直接書き込み
- **uflash コマンド**: `uflash main.py` でコマンドラインから転送

## 使用ライブラリ

すべて MicroPython の標準ライブラリです。追加のインストールは不要です。

| ライブラリ | 用途 |
|---|---|
| `microbit` | ボタン入力、LED 表示、スリープ |
| `music` | 音符の再生 |

## 制約事項

- **単音再生のみ**: micro:bit の music モジュールは同時に1音しか鳴らせません。和音やドラムパートには対応していません
- **メモリ制限**: micro:bit の RAM は約 16KB（v1）/ 128KB（v2）です。長大な曲データを複数入れるとメモリ不足になる可能性があります
- **音質**: 圧電ブザー/内蔵スピーカーによる矩形波出力です。楽器のような音色は出ません
